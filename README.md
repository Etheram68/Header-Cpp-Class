# 42 Lyon/Paris Canonical Class for Cpp

## Install

Launch Quick Open with <kbd>⌘</kbd>+<kbd>P</kbd> and enter
```
ext install canonicalclasscpp
```

## Usage
### Insert a header
 - **macOS** : <kbd>⌘</kbd> + <kbd>⌥</kbd> + <kbd>C</kbd>
 - **Linux** / **Windows** : <kbd>Ctrl</kbd> + <kbd>alt</kbd> + <kbd>C</kbd>.
